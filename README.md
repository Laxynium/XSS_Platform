To run application you need to:
- Run docker with level (docker-compose up in Levels/LevelX)
- Run main app (npm start in 'XSS-Platform' folder)

To run Unit tests:
- npm test

To run UI tests:
- npm run e2e:ui or npm run e2e:uidev
